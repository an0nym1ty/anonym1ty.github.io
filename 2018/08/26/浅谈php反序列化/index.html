0x02 利用构造函数

Magic function

php中有一类特殊的方法叫”Magic function”，这里着重关注以下几个：

  • 构造函数__construct()：当对象创建(new)时会自动调用。但在unserialize()时是不会自动调用的。

  • 析构函数__destruct():当对象被销毁时会自动调用。

  • __wakeup():unserialize()时会自动调用。
